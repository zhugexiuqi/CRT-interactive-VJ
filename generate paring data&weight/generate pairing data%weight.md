#generate pairing data & weight
这个文件夹里是MATLAB脚本，及其生成的将要被我们用于配置FPGA的文件。

我们的任务是将1080p的视频，即分辨率为1920* 1080的视频缩小到VGA协议的 640* 480分辨率。  
注意图像分辨率a* b是列数在前，行数在后。1080分辨率意为1080行。

我们采用**双线性插值法**来处理图像缩放。

首先，将缩放后图像的像素坐标**逆映射**到原图像大小。例如，1080/480 = 2.25，我们将0-479的缩放后像素坐标
均乘以2.25。该过程即逆映射。缩放后像素坐标将落入原图坐标的两个整点区间内，例如2*2.25 = 4.5，落入(3,4)区间。

实际的视频信号以**行扫描流式**输入，我们需要存储区间后点的值；这样，当某个区间的后点的像素值传送完成后，
就开始对最近输入的两个像素点的值做插值计算。
